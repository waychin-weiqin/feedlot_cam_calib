\section{Method}
\label{sec:method}

Our method consists of four stages, each building on the previous.

\subsection{Stage 1: Baseline Pipeline}

Given $K=91$ timesteps of 8-camera images, we run Pi3X to obtain per-timestep poses $\{T_k^c\}_{k=1}^K$ for each camera $c \in \{1,\ldots,8\}$, where each $T_k^c \in \text{SE}(3)$ is defined up to a per-timestep Sim(3) gauge.

\paragraph{Gauge fixing.} We align all timesteps to a reference frame via Sim(3) Procrustes (Umeyama's method), removing the per-timestep scale and frame ambiguity.

\paragraph{Outlier rejection.} For each camera, we project its $K$ estimated positions onto the dominant 2D plane (via PCA) and reject outliers using the Median Absolute Deviation (MAD) with threshold $k=3$.

\paragraph{Pose graph optimization.} We extract pairwise relative poses from inlier timesteps, then optimize globally via:
\begin{itemize}
    \item $L_1$ Weiszfeld rotation averaging on SO(3)
    \item Translation averaging with hard co-location constraints (cameras sharing a pole are forced to the same position)
\end{itemize}

\subsection{Stage 2: Self-Conditioned Inference}

Pi3X supports multimodal conditioning: given poses $\hat{T}^c$ and intrinsics $K^c$, it refines its predictions. We condition on the pipeline output from Stage~1:
\begin{equation}
    \{T_k^{c,(1)}\} = \text{Pi3X}(\text{imgs}_k \mid \hat{T}^c, K^c), \quad k = 1, \ldots, K
\end{equation}
where $\hat{T}^c$ is the Stage~1 estimate converted to video camera order.

\subsection{Stage 3: Hybrid Fusion}

We observe that conditioned inference improves rotation but degrades translation (Section~\ref{sec:ablation}). To exploit both, we fuse edges from both sources in the pose graph:

\begin{equation}
    \tilde{T}_{ij}^{\text{fused}} = \begin{pmatrix} R_{ij}^{\text{cond}} & t_{ij}^{\text{uncond}} \\ 0 & 1 \end{pmatrix}
\end{equation}

where $R_{ij}^{\text{cond}}$ is the mean relative rotation from conditioned inference and $t_{ij}^{\text{uncond}}$ is the mean relative translation from unconditioned inference. Edge weights are taken as $w_{ij} = \max(w_{ij}^{\text{cond}}, w_{ij}^{\text{uncond}})$.

The initial pose for optimization uses conditioned rotations with unconditioned translations:
\begin{equation}
    T_{\text{init}}^c = \begin{pmatrix} R_{\text{cond}}^c & t_{\text{uncond}}^c \\ 0 & 1 \end{pmatrix}
\end{equation}

\subsection{Stage 4: Monte Carlo Iterative Refinement}

To further improve rotation estimates, we iterate the hybrid fusion with stochastic perturbation:

\begin{algorithm}[h]
\caption{Iterated Hybrid MC Refinement}
\label{alg:itermc}
\begin{algorithmic}[1]
\REQUIRE Unconditioned pipeline output $\hat{T}^c$, images, intrinsics
\REQUIRE Noise parameters $\sigma_R, \sigma_t$; samples $S$; iterations $M$
\STATE $T_{\text{current}} \leftarrow \hat{T}^c$
\STATE Compute unconditioned translation edges $\{t_{ij}^{\text{uncond}}, w_{ij}^t\}$
\FOR{$m = 1$ to $M$}
    \FOR{$s = 1$ to $S$}
        \STATE $T_s \leftarrow \text{AddNoise}(T_{\text{current}}, \sigma_R, \sigma_t)$
        \STATE $\{T_{k,s}^c\} \leftarrow \text{Pi3X}(\text{imgs} \mid T_s, K^c)$ for all timesteps
        \STATE $\hat{T}_s^c \leftarrow \text{Pipeline}(\{T_{k,s}^c\})$ \COMMENT{gauge fix + outlier rej}
    \ENDFOR
    \STATE Extract rotation edges $\{R_{ij}^{\text{cond}}, w_{ij}^R\}$ from all $S \times K$ aligned poses
    \STATE Fuse: $\tilde{T}_{ij} = (R_{ij}^{\text{cond}}, t_{ij}^{\text{uncond}})$
    \STATE $T_{\text{opt}} \leftarrow \text{PoseGraph}(\bar{T}_{\text{init}}, \tilde{T}_{ij}, \max(w^R, w^t))$
    \STATE $T_{\text{current}} \leftarrow (R_{\text{opt}}, t_{\text{uncond}})$ \COMMENT{preserve translations}
\ENDFOR
\RETURN $T_{\text{opt}}$
\end{algorithmic}
\end{algorithm}

Key design choices:
\begin{itemize}
    \item \textbf{Noise}: Gaussian perturbation with $\sigma_R = 2°$ (axis-angle) and $\sigma_t = 0.05$ (isotropic)
    \item \textbf{Translation anchoring}: Translations are always sourced from the unconditioned baseline, never updated from conditioned outputs
    \item \textbf{Early stopping}: Monitor ARE convergence; terminate when improvement $< 0.1°$
\end{itemize}
